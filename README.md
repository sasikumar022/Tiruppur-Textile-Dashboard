🧵 Tiruppur Textile Export Analytics (2014–2024)

live link:https://sasikumar022.github.io/Tiruppur-Textile-Dashboard/

🌟 Project Vision
As a BCA student at SASTRA University, I developed this project to demonstrate how data analytics can be applied to niche regional industrial clusters. This dashboard is not just a visualization; it is a Macro-Economic Story of Tiruppur’s resilience. By integrating data from the Ministry of Textiles and DGCI&S, I mapped out a decade of trade dynamics to identify risks and growth opportunities in the global knitwear market.

🧠 Strategic Development (AI-Assisted)
This project was developed using a Human-in-the-Loop AI workflow. While the frontend and logic were generated via Claude AI, the strategic direction was entirely human-led
Domain Direction: Defined the specific KPIs relevant to the Tiruppur cluster (EU share, China+1 impact).

Data Logic: Orchestrated the logic for calculating 5-year CAGR and YoY growth rates.

Prompt Engineering: Iteratively refined complex CSS "Dark Mode" aesthetics and Chart.js configurations to meet professional analyst standards.

🛠️ Tech Stack
Visualization: Chart.js (Interactive Time-Series, Heatmaps, and Correlation plots).

Core Logic: JavaScript (ES6+) for dynamic filtering and "What-If" event toggles.

Styling: CSS3 with custom "Glassmorphism" and "Grain" overlays for a premium executive feel.

Analytics: SQL-based logic for CAGR and Pearson Correlation (r = +0.72) between INR/USD and Export Value.

📈 Key Insights Highlighted
EU Over-Dependence: Identifying the 58.2% market concentration risk.

The China+1 Pivot: Visualizing the 52% surge in US market share since 2021.

Seasonal Intelligence: Highlighting the "Golden Window" (Oct–Feb) where 62% of annual revenue is generated.
